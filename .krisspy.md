# Project Instructions

This is a **React + TypeScript + Tailwind CSS** project running in Sandpack.

## Tech Stack

- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS for utility-first styling
- **Icons**: Lucide React for beautiful, customizable icons
- **Routing**: React Router DOM for page navigation
- **Images**: Unsplash for high-quality placeholder images

## Quick Start

### Using Icons
Import icons from lucide-react:
```tsx
import { Heart, Star, User, Settings } from 'lucide-react';

<Heart className="w-6 h-6 text-red-500" />
<Star size={24} color="gold" />
```

## Development Tips

1. **Hot Reload**: Changes are applied instantly in Sandpack
2. **TypeScript**: Full TypeScript support with type checking
3. **Responsive Design**: Use Tailwind's responsive prefixes (`sm:`, `md:`, `lg:`)
4. **Icons**: Browse all available icons at [lucide.dev](https://lucide.dev)
5. **Images**: Generate random images with [picsum.photos](https://picsum.photos)
